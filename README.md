# 📖 Clube do Livro

Projeto de gerenciamento de leituras feito com Flask e PostgreSQL.

## Como rodar

Instale as dependências:

```bash
pip install -r requirements.txt
```
Crie um banco PostgreSQL chamado:

```bash
clube_do_livro
```
Depois execute os arquivos:

```bash
sql/schema.sql
sql/seed.sql
```

Logo em seguida rode:

```bash
cd app
python app.py
```
